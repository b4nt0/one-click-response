# Quick response block design requirements

## Legibility

The response block must be legible in light mode and dark mode at the recipient's side.

## No built-in text

The response block must not assume the recipient's language and add any headers, titles, and other common text. The only text rendered must be the button captions determined by the user.

## Editable

The response block must not use any complex formatting. The goal is for the block to be editable using standard Gmail tools. If the user wants to move the buttons around or delete the block entirely, it should be easily done.

## Compact

The button options must be rendered in a row with a margin. The button row must wrap automatically to the available screen width.
