# Warnings during adding a quick response block

Adding response buttons features the following warnings and notes.

## Multiple recipient warning

Where: Under the "I confirm these recipients are correct" checkbox.
When: When there is more than one recipient of the email. The state of the confirmation checkbox does not matter.
What: "You are sending to many recipients. They can only give one answer per group, only the first answer will count".

## Don't copy warning

Where: Under the "I confirm these recipients are correct" checkbox.
When: When the confirmation checkbox is ON
What: "Don't copy the response buttons to other emails".
